# 生命の夜 — A Night of Life

**日本語** | [English](README.en.md)

遺伝的アルゴリズムで進化し続ける、光のアート。

> 生命は、夜に閉ざされている。この箱庭の中では、15種類の遺伝子を持つ光が
> 遺伝的アルゴリズムに従い世代を重ね変化する。その行き着く先で、
> 偶然的なものを読み取るか、必然的なものを読み取るかは、観察者に委ねられている。

画面のなかを、いくつもの「光の種族」が静かに流れています。
種族は世代交代をくり返し、色も動きも少しずつ移ろい、
ときおり突然変異で、これまでに無い色の一族が生まれます。
同じ眺めは、二度と訪れません。

## 見る

ブラウザで開くだけ: https://hello-amedev.github.io/ART/night-of-life/

## 壁紙として使う(Windows・Lively Wallpaper)

1. [Lively Wallpaper](https://www.rocksdanister.com/lively/) をインストールして開く
2. 「＋」(壁紙の追加)→ 同梱の `index.html` を選ぶ
3. 壁紙を右クリック →「カスタマイズ」で見た目を調整

## カスタマイズでできること

| 項目 | 内容 |
|---|---|
| 粒子の量 | 光の数。重く感じたら減らす |
| 世代交代の間隔(分) | 進化のテンポ |
| 明るさ(%) | 全体の明るさ |
| 軌跡の長さ(%) | 光の尾の長さ |
| 省電力モード | 描画を軽くする |
| システム表示(世代・種族) | 右下の小さな計器の表示/非表示 |
| 突然変異を起こす | いますぐ突然変異を起こす |
| 進化をリセット | 最初からやり直す |
| 下端の余白 | 計器がタスクバーに隠れるときに調整 |

## 観測パネルの見かた(右下)

無機質な観察者の計器のように、いま画面にいる光の種族を 1 行ずつ映します
(「システム表示」で消せます)。

- 上段 … 現在の世代と時刻
- 色チップ … その種族の体色(画面の光と見比べられます)
- `gen` / `from` … 世代と、掛け合わせの親(例: 42 / 38×35)
- `act` … いまの活発さ(その時刻に元気な種族ほど高い)
- `vig` … 家系の勢い(子を残し続ける家系は高く、途絶えると衰える)
- `spd` / `flk` / `len` … 速さ・群れ度・光の長さ
- `age` … 生きてきた時間(分:秒)
- 右端 … 状態(in 誕生中 / ↓out 退場中 / ✦ 突然変異)
- 下段のログ … 直近の誕生(↑)・退場(↓)・突然変異(✦)・新規参入(+)

## 知っておくと良いこと

- 進化は自動で進み、PC を再起動しても続きから育ちます
- 同じ模様は二度と現れません
- 壁紙ファイルを新しくしたら、Lively で一度削除して入れ直してください
