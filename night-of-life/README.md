# 生命の夜 — A Night of Life

遺伝的アルゴリズムで進化し続ける光の芸術作品。Lively Wallpaper 用の壁紙としても配布する。
箱庭を眺めるように、そこで起こる変化・無常さ・受け継がれていくものを感じ取る作品。

> 生命は、夜に閉ざされている。この箱庭の中では、遺伝的アルゴリズムによる光が
> 世代を越えて変化する。その行き着く先で、偶然的なものを読み取るか、
> 必然的なものを読み取るかは、観察者に委ねられている。
>
> Life is closed within the night. In this small garden, light born of a genetic
> algorithm changes across the generations. At where it arrives, whether you read
> the accidental or the inevitable is left to the observer.

無数の「光の筆致(短い光の針)」が、目に見えない風に乗って
ゆっくりと画面を流れる。筆致の「種族」が色・動き・揃い方・
活動時間帯の遺伝子を持ち、時刻を選択圧として世代交代していく。
ときおり「突然変異」が起き、親とはまるで違う色の一族が、ひときわ輝いて生まれる。
1 日の時の流れとともに色彩と構図が移ろい、再起動しても続きから育ち続ける。

デザインの軸は「無機的なものが有機的に動く」。
個々の粒は硬質でまっすぐな光の針(生き物ではない)、
流れ場が生む集団の動きだけが有機的な美しさを作る。

## 動かし方

### Web で見る(公開ページ)

https://hello-amedev.github.io/ART/night-of-life/

ブラウザで開くだけ。マウスを動かす(スマホは画面に触れる)と、右上に設定パネルが
淡く現れます(粒子の量・明るさ・軌跡・突然変異を起こす など)。しばらく無操作だと
静かに消えます。進化は実時間で進むので、長く眺めるほど世代交代や突然変異に出会えます。

### ローカルで見る

index.html をダブルクリックするだけ(ビルド・サーバー不要)。公開ページと同じものです。
公開ページ・通常表示はいずれも**実時間で進化**します。以下は開発用のオプションです。

- 進化を早回しで見る(開発用): URL の末尾に `?demo`(1 日が約 3 分で回る)
- 内部状態を見る(開発用): `?debug`(左下に世代と種族の状態が出る)
- 早回しの進化は保存されません(本物の進化を汚さないため)

### Lively Wallpaper に入れる(デスクトップの壁紙として)

1. Lively Wallpaper を開く
2. 「+」(壁紙の追加)をクリック
3. このフォルダの index.html を選ぶ
4. 壁紙を右クリック → カスタマイズ で設定を調整

## 設定項目(Lively のカスタマイズ画面)

| 項目 | 内容 |
|---|---|
| 粒子の量 | 光の粒の総数。重いと感じたら減らす |
| 世代交代の間隔(分) | 進化のテンポ。短いほど移ろいが速い |
| 明るさ(%) | 全体の輝度 |
| 軌跡の長さ(%) | 光の尾の長さ |
| 省電力モード | 30fps に間引いて描画負荷を半分にする |
| システム表示(世代・種族) | 右下の観測パネルの表示/非表示 |
| 下端の余白 | 観測パネル等を持ち上げる量。タスクバーに隠れる場合に調整(既定 60) |
| 突然変異を 1 回起こす | いますぐ突然変異を起こす(親とまるで違う色の一族が生まれる)。確認用 |
| 進化をリセット | 第 1 世代からやり直す |

Web 版(公開ページ)では、このうち主要な項目(粒子の量・世代交代の間隔・明るさ・
軌跡の長さ・省電力・システム表示・突然変異を起こす・進化をリセット)を右上のパネルで
調整できる。Lively 専用の項目(下端の余白・診断)は Web 版には出ない。

Lively の壁紙はタスクバーの裏まで描画されるため、画面下端基準の表示は
「下端の余白」で持ち上げている。ブラウザで開いた場合は余白 0 で表示される。

注意: Lively は壁紙を追加した時点のファイルをコピーして使う。
このフォルダのファイルを更新したら、Lively 側の壁紙を削除して入れ直すこと。

## 観測パネル(右下のシステム表示)

無機的な観察者の計器のように、種族ごとに 1 行、進化の状態を素の数値で並べる。

```
A NIGHT OF LIFE   gen 42 · 19:42
     gen from   act  vig  spd  flk  len   age
■    42  38×35  .67  .80  .31  .15  .57  3:12   ·
...
↑ 43 ‹40×38  +2s
↓ 39         +2s
✦ 41 ‹38×40  +41s
```

- 色チップ: その種族の体色(画面の光の線と見比べられる。常に純色で表示)
- `gen` / `from`: 今の世代と、掛け合わせの親(系譜)。例 42 / 38×35
- `act`: いまの活発さ(目覚め度)。夜行性の種族は夜に上がる
- `vig`: 家系の勢い。子を残し続ける家系は高く、途絶えると衰えて消える
- `spd` / `flk` / `len`: 速さ・群れ度・筆の長さ(画面の動きと見比べられる)
- `age`: 生きてきた齢(分:秒)
- 右端: 状態(誕生中 in / 退場中 ↓out / 突然変異の印 ✦)
- 下の事象ログ: 直近の誕生(↑)・退場(↓)・突然変異(✦)・参入(+)を経過秒つきで

新しく生まれた行は、受け継いだ色でうっすら灯ってから馴染む(相続の可視化)。

## 進化の見どころ

- 世代交代の瞬間、新種族の光は「親 2 種族の流れの中」から湧き出してくる。
  どこからともなく現れるのではなく、親の位置から生まれる(交叉の可視化)
- ときおり「突然変異」が起こると、親とはまるで違う冴えた色の一族が生まれる。
  派手な光の演出はあえて無くし、変異は「これまでに無い色」そのもので気付かせる。
  計器にはしばらく ✦ の印が灯る
- 退場する種族は急に消えず、だんだん動きが鈍り、輝きを失いながら消えていく
- 時間帯が変わると、その時刻が得意な種族の光が強まり、苦手な種族は静かになる。
  観測パネルのバーと画面を見比べると対応が分かる

## 進化のしくみ(ざっくり)

- 画面には常時 5 つの「種族」がいて、それぞれが遺伝子
  (色相・彩度・光の太さ・速度・風への従順さ・群れ行動・活動時間帯など)を持つ
- 一定間隔で世代交代が起こる:
  いまの時刻に最も合わない種族が静かに消え、
  合っている 2 種族の遺伝子を交叉 + 突然変異した新種族が生まれる
- 朝型・夜型のすみ分けが生まれるよう、活動時間帯が他種族と被ると不利になる。
  さらに色が他種族と近くても不利になるので、画面は色とりどりに保たれる(ニッチ分化)
- 活動時間が狭い種族はピーク時に強く輝き、広い種族は常にほんのり輝く
  (スペシャリスト / ジェネラリストのトレードオフ)
- 進化の状態は WebView の保存領域(localStorage)に記録され、
  PC を再起動しても続きから育つ

## ファイル構成

| ファイル | 役割 |
|---|---|
| index.html | 本体 |
| js/noise.js | ノイズ(風の元になる滑らかな乱数) |
| js/field.js | 風の地図(カールノイズのフローフィールド) |
| js/genome.js | 遺伝子の定義・交叉・突然変異・適応度 |
| js/species.js | 種族と粒子(群れ行動・移動・フェード) |
| js/evolution.js | 世代交代・時刻の選択圧・保存/復元 |
| js/lively.js | Lively 連携と実行時設定 |
| js/main.js | 描画ループ・時刻→色変換 |
| js/webui.js | Web 版の設定パネル(マウスで現れる。Lively では出さない) |
| LivelyProperties.json | Lively の設定画面の定義 |
